# Code Availability Statement

The code is available as an LLM-guided, schema-constrained CAR-T in silico workflow. The deterministic reference mode is the default and can be run without API keys, internet access, or a compiled PhysiCell executable.

Optional modes include executable LLM-agent mode, archived/replay mode, hybrid LLM plus deterministic validation mode, ablation analysis, and external PhysiCell execution.

The repository does not commit:

- fabricated citations
- fabricated LLM outputs
- fabricated PhysiCell outputs
- fabricated wet-lab values
- compiled PhysiCell binaries
- large generated simulation outputs

Mock records are software fixtures only. They are not real scholarly citations and are not manuscript evidence.
