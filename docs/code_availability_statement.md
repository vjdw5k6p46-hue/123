# Code Availability Statement

The code is available as an LLM-guided, schema-constrained CAR-T in silico workflow. The AutoResearch entry point is LLM-first when provider credentials are configured, while the deterministic reference mode remains available and can be run without API keys, internet access, or a compiled PhysiCell executable.

Optional modes include executable LLM-agent mode, hybrid LLM plus deterministic validation mode, ablation analysis, and external PhysiCell execution. Executable LLM runs save prompt-response audit artifacts for inspection.

The repository does not commit:

- fabricated citations
- fabricated LLM outputs
- fabricated PhysiCell outputs
- fabricated wet-lab values
- compiled PhysiCell binaries
- large generated simulation outputs

Mock records are software fixtures only. They are not real scholarly citations and are not manuscript evidence.
